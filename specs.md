# Simple Color Changer Webpage

## Goal

Build a simple webpage that demonstrates basic HTML, CSS, and JavaScript.

## Functional Requirements

- Display the title "Hello GitHub Copilot!"
- Show a short welcome message.
- Display a button labeled "Change Background Color".
- Clicking the button should cycle through at least four background colors.
- The page should be centered.
- The design should be responsive.
- Do not use any JavaScript or CSS frameworks.

## Non-functional Requirements

- Use semantic HTML.
- Keep JavaScript in a separate file.
- Keep CSS in a separate file.
- Use modern ES6 syntax.
- Keep the code simple and beginner-friendly.

## Deliverables

- index.html
- styles.css
- script.js
- README.md
