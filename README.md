# Hello Webpage Spec

A simple, beginner-friendly webpage built with plain HTML, CSS, and JavaScript.

## Features

- Displays the title: Hello GitHub Copilot!
- Shows a short welcome message
- Includes a button to change background color
- Cycles through multiple background colors on each click
- Centers content on the page
- Uses a responsive layout

## Files

- index.html
- styles.css
- script.js
- README.md

## How to Run

1. Open `index.html` in your browser.
2. Click the **Change Background Color** button to cycle through colors.
